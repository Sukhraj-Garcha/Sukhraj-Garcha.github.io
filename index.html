Cesar Cypher Decoder

This project was programmed using C. The program reads in a file that contains structures that each have two strings and two arrays. The data in the file is in binary so the program reads it and displays it as readable characters. The strings are also encoded using a cesar cypher and if specified, the strings are decoded using the chi_squared formula.
